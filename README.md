# 🕋 Prayer Times Web Application

A dynamic and highly responsive **Prayer Times** web application built using **React.js**, **Vite**, and styled with **Tailwind CSS**. This application fetches accurate real-time prayer timings based on the user's location or selected city, providing a clean and spiritual user experience.

## 💻 Tech Stack

*   **Frontend Library:** React.js (Functional Components & Hooks)
*   **Build Tool:** Vite (For ultra-fast development and bundling)
*   **Styling:** Tailwind CSS (Modern utility-first CSS framework for elegant layouts)
*   **API Integration:** Fetching real-time astronomical data from reliable Prayer Times APIs.
*   **Deployment:** Netlify (Continuous Deployment linked with GitHub)

---

## ✨ Features

*   **Real-time Prayer Schedule:** Displays accurate daily timings for Fajr, Sunrise, Dhuhr, Asr, Maghrib, and Isha.
*   **Dynamic City/Location Selection:** Allows users to change cities or regions to update countdowns and timings instantly.
*   **Live Countdown Timer:** Features an interactive countdown clock showing the remaining time until the next upcoming prayer.
*   **Hijri & Gregorian Dates:** Displays integrated, auto-updating dates for both calendars.
*   **Fully Responsive & Elegant UI:** A clean, peaceful, and mobile-friendly interface designed with customized Tailwind UI layouts.
